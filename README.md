# 🧥 ROMP — Premium Oversized Hoodie Store

ROMP is a modern streetwear-inspired hoodie brand website designed with a clean UI, smooth layout, and responsive design.  
This project is built using HTML, CSS, and JavaScript and deployed using GitHub Pages.

---

## 🌐 Live Demo

👉 https://iqra-a7.github.io/ROMP


---

## 🧠 About the Project

ROMP is a fashion landing page concept focused on:
- Minimal premium design
- Streetwear hoodie branding
- Strong visual hierarchy
- Smooth user experience
- Mobile-friendly responsive layout

This project is great for portfolio and frontend practice.

---

## ✨ Features

- Modern hero section design
- Product showcase cards
- Smooth hover effects
- Interactive UI elements
- Fully responsive layout
- Fast loading static website
- Clean and organized code

---

## 🛠️ Tech Stack


- HTML5  
- Tailwind CSS (CDN)  
- JavaScript (DOM manipulation)  
- Framer Motion (CDN animation effects)  
- GitHub Pages (Hosting)
